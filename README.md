Este proyecto fue desarrollado para el trabajo final del curso de Javascript (Comision 88225) de CODORHOUSE.Este trbajo Consiste en una página web desarrolada para mi emprendimiento Patagonia Go Play (PGP),donde el usuario puede navegar por el catálogo de juegos y agregarlos a un carrito de compras.El objetivo principal del proyecto es demostrar el uso integrado de HTML, CSS y JavaScript para construir una interfaz funcional y organizada.
Y asi a su vez ya poder desarrolar la parte del catalogo y carrito,para poder integrar con la pagina web creada en el curso de Desarrollo Web (Comision 87900) de CODERHOUSE para la pagina de mi emprendimineto de venta de videojuegos para diferentes paltaformas 

La estructura del sitio está compuesta por dos páginas principales: index.html que funciona como la página de inicio y muestra los productos disponibles, y luego esta carrito.html donde se visualizan los elementos que el usuario fue agregando al carrito.

El diseño visual del proyecto se maneja mediante styles.css que contiene todos los estilos aplicados a los distintos elementos de la interfaz,asegurando coherencia estética y una correcta distribución del contenido. 
Por otro lado la lógica del carrito y el comportamiento interactivo del sitio están implementados dentro de la carpeta /js,donde se administran acciones como agregar productos,verificarlos y actualizarlos.Además el proyecto incluye una carpeta assets/img la cual almacena las imágenes utilizadas en la página y una carpeta data destinada a contener los datos necesarios para el funcionamiento del catálogo o configuraciones adicionales.

Con esto ademas de lograr esto para mi pagina web busco mostrar en este trabajo fianl como se puede organizar un proyecto web de forma limpia manteniendo separadas la estructura,el diseño y la lógica.

*Para poder visualizar la pagina en su totalidad se necesita abrir desde el live serve en VS (Visual Studio Code),ya que no esta subida a un servidor.
